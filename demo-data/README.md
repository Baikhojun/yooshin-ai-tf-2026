# 강사 시연용 가공 데이터

> (주)멋진엔지니어링 관리부를 무대로 한 1회차 강의 시연·실습 자료.
> **모두 가공 데이터입니다 — 실제 인물·실제 회사·실제 데이터 없음.**

## 파일 목록

| 파일 | 용도 | 사용 위치 (본문 기준) |
|------|------|---------------------|
| `휴가신청_사유_50건.txt` | 실습 ② 의미공간 분류 데이터 | Part 2.6 |
| `회의메모_샘플_3건.txt` | 라이브 시연용 회의 메모 | Part 1.4, Part 4.4.3 |
| `회의록_정리.md` | 강사 견본 *MD 프롬프트 파일* | Part 4.4.3 시연 |
| `빈_템플릿.md` | 위원에게 배포할 *빈 6칸 템플릿* | 과제 ③ |

## 사용 방법

### 1. 휴가신청_사유_50건.txt
- 메모장 또는 엑셀로 열어서 위원에게 보여줌
- 위원이 전체 복사 → ChatGPT 새 대화에 붙여넣기 → *"의미별로 묶어 표로 정리해줘"* 한 줄

### 2. 회의메모_샘플_3건.txt
- 3개 샘플 중 *1건만* 강의에서 시연 (나머지는 강사 노트로)
- 가장 친근한 **샘플 1 (부서 운영 회의)** 추천

### 3. 회의록_정리.md
- Part 4.4.3 라이브 시연의 *주인공*
- 강사가 메모장으로 *직접* 띄워서 위원에게 *"이게 그냥 텍스트 파일입니다"* 강조
- ChatGPT 새 대화에 *전체* 복사·붙여넣기 → 맨 아래 `[데이터]` 영역에 회의메모 추가
- 결과 확인 후, 새 채팅에서도 같은 결과가 나오는지 *재현성* 보여줌

### 4. 빈_템플릿.md
- 과제 ③ (선택) 으로 배포
- 위원이 본인 업무용으로 *복사해서 채워* 보는 출발점

## 보안 / 가공 정책

- 회사명은 모두 **(주)멋진엔지니어링** (가공명)
- 부서명은 *관리부 / 인사부 / 총무부 / 재무부 / 자산팀* (실제 회사 부서명 미반영)
- 인명은 *김 부장*, *박 차장* 같은 *직급 호칭* 또는 **흔한 가공 한국 성씨+이름** 조합 — 특정 개인 매핑 불가
- 휴가 사유는 *직장인 일상에서 흔히 발생*하는 패턴만 (실제 사례 차용 없음)
- 금액·날짜는 모두 가공
